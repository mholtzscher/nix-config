tmux Cheatsheet for Zellij Muscle Memory

This setup intentionally keeps tmux close to its defaults. The main mental shift is moving from Zellij modes to tmux's prefix model.

Remember: prefix means press Ctrl-b, release, then press the next key.

Plugin mapping

Catppuccin

catppuccin.tmux.enable = true keeps theme management in the existing Catppuccin flake/module.

tmux-resurrect

Manual save/restore for sessions, windows, panes, layouts, and commands.

tmux-continuum

Periodic session saves and optional restore behavior via plugin defaults/config.

vim-tmux-navigator

Ctrl-h/j/k/l navigation between Vim splits and tmux panes, assuming editor-side config is present.

tmux-yank

Easier copy-mode yanking to the system clipboard.

tmux-sensible

Loaded through sensibleOnTop = true for community-standard tmux defaults.

Current minimal config summary

  • tmux
  • mouse support
  • tmux sensible defaults
  • Catppuccin tmux theme via the Catppuccin module
  • resurrect, continuum, vim-tmux-navigator, and yank

Everything else is intentionally left as tmux default behavior.
